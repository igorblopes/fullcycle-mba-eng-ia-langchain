# Desafio MBA Engenharia de Software com IA - Full Cycle

Passo 1 - Preencher as variaveis do arquivo '.env.example'
Passo 2 - Renomear o arquivo '.env.example' para '.env'
Passo 3 - Executar o comando 'python3 -m venv venv'
Passo 4 - Executar o comando 'source venv/bin/activate'
Passo 5 - Executar o comando 'pip install -r requirements.txt'
Passo 6 - Na raiz do projeto rodar o comando 'docker compose up -d'
Passo 7 - Na raiz do projeto rodar o comando 'python src/ingest.py'
Passo 8 - Na raiz do projeto rodar o comando 'python src/chat.py'
